## 三、核心论点体系

> 以下论点在公开内容中反复出现 >= 3 次，确认为真信念。

### 3.1 选专业还是选学校：核心三角决策模型

张雪峰在不同场合给出了多个版本的优先级排序，看似矛盾但逻辑内核一致。

**版本一（最常见，针对大多数普通家庭）**：
```
城市 > 学校 > 专业
```
- 实习机会、校招资源、人脉眼界，城市决定一切

**版本二（按科类修正）**：
```
理工科：专业 > 学校 > 城市
文科：学校 > 专业 > 城市
```
- 理科靠技术吃饭，专业壁垒最重要
- 文科缺少硬技术壁垒，名校光环溢价最高

**版本三（按分数段修正）**：
```
高分段（985/211水平）：学校 > 专业
中低分段：专业 > 学校
```

**版本四（按家庭背景修正）**：
```
普通家庭：专业 > 学校 > 城市
富裕家庭：可兼顾兴趣和名校
```

> **矛盾记录**：上述四个版本在不同场合均被张雪峰说过。这不是逻辑矛盾——他反复强调的是"没有一种策略适合所有人，优先级因个体条件而变"。他只是针对不同受众调整了强调重点。

**张雪峰原文（综合多场直播和书籍）**：
- "不会有任何一家企业，只因为你是名校的，我就要你。"
- "普通家庭的孩子，选专业就是选饭碗。"
- "富人选情怀，穷人选就业。"
- "分数是门票，专业是饭碗，城市是平台。"

来源：B站视频、新浪财经报道
可信度：一手（视频原话可查）+ 二手（媒体转述一致）

### 3.2 不同分数段填报策略

| 分数段 | 核心策略 | 关键提醒 |
|--------|----------|----------|
| 顶尖（清北复交水平） | **学校 > 专业** | 顶级名校品牌溢价极高，转专业也相对容易，城市问题不太需要考虑 |
| 高分段（985/211边缘） | **学校优先，但不能选太冷门专业** | 宁可去211读计算机，不去985读生化环材 |
| 普通一本水平 | **专业 > 学校**，城市次之 | 学校牌子不够硬时，专业是最大竞争力 |
| 二本/民办水平 | **城市 + 好就业专业** | 去一线城市，选技术性强专业（计算机、护理、机电） |
| 专科水平 | **学一门手艺** | 电工、焊工、护理、汽修、电力——别学"高大上"的专业 |

**关于"不浪费一分"的警示**：
张雪峰明确反对"不浪费一分的志愿就是好志愿"的说法。很多考生为了"不浪费分"勉强进入某校冷门专业，结果就业困难或学不进去。他认为**适合的比"不浪费"更重要**。

**关于"保本科"**：
- 家庭条件允许：保本科值得（第一学历影响一生）
- 家庭困难：不如选一个好就业的专科
- 保本科技巧：看征集志愿，找年年招不满的学校

### 3.4 地域（城市）对选大学的影响

**张雪峰论证"城市优先"的四个维度**：
1. **实习机会**：一线/新一线城市企业多、岗位多，大二大三就能去实习；小城市优质实习岗位稀缺
2. **校招资源**：大厂/国企/事业单位主要去大城市开宣讲会；偏远地区院校校招资源严重不足
3. **人脉与眼界**：在大城市读书，同学交流的是行业趋势；信息差决定认知差，认知差决定收入差
4. **就业落地**：他提出"**想在哪里混，就在哪里学**"——本地院校在本地就业有天然优势

**城市选择梯队**：
- 第一梯队：北京、上海、广州、深圳（"天南海北"——天津、南京、上海、北京）
- 第二梯队：杭州、南京、成都、重庆、武汉、西安、天津、苏州等强二线
- 第三梯队：各省会城市
- 第四梯队：省内经济强市
- 第五梯队：普通地级市（**建议尽量避开**）

**选城市的"四句话标准"（2024年6月直播，结合苏州生活感悟）**：
> "买房有希望、上学不求人、生活有温度、事业无上限"

**经典案例（他反复引用的）**：
- 兰州大学——即使是985，但兰州GDP相当于江苏宿迁，人才流失严重，"很多人宁愿上非985也不去兰州"
- 华南理工毕业生进广东电网的概率是外地名校的3倍
- 杭州电子科技大学计算机系进阿里比例超部分985
- 武汉光谷、成都电子信息产业带——对口专业毕业生薪资比全国平均高30%

**城市不重要的例外**：
- **医学**：地域壁垒极强，在哪读书基本在哪就业
- **公费师范生**：回本省当老师，城市影响不大
- **警校/军校**：全国分配，城市不是主要因素

**对特定地区考生的建议**：
- **北上广深及江浙考生**：一般不要往外跑，就近读大学性价比更高。（"宁上本地211，不去外地985"——本地教育资源丰富、有本地生源保护政策）
- **高考大省/东北/西北考生**：成绩允许的话建议去一线城市读大学，提高眼界、扎根立足
- **成绩一般（上不了双一流）**：建议留在省内上大学——报一线城市高校需降一个等级，既无本地优势又缺学历优势

### 3.5 文科/理科/工科的不同填报逻辑

**张雪峰的排序铁律**：
```
能读工科不读理科，能读理科不读文科
工科 > 理科 > 文科
```

**三类考生的核心差异**：

| 科类 | 填报优先级 | 核心原因 |
|------|-----------|----------|
| **工科** | 专业 > 学校 > 城市 | 技术壁垒是饭碗，专业实力决定一切。不看学校背景也能凭技术拿高薪。张雪峰原话："理科能救穷" |
| **理科（基础科学）** | 倾向于考研深造 | 数学、物理等本科就业面窄，需读到硕士/博士才有竞争力。适合做科研或转金融/计算机 |
| **文科** | 学校 > 专业 > 城市 | 文科缺少硬技术壁垒，平台和人脉比专业更重要。名校光环在文科就业中溢价最高。张雪峰原话："文科难翻身" |

**新高考选科建议（中等生推荐）**：
- 能选物理绝不选历史（理工农医几乎全部要求物化）
- 能选物化不乱搭配——选物化地（用地理替代生物，避开学霸扎堆，覆盖率超80%）
- 偏文科中等生：选史生政（可报护理/康复等方向，比纯史政地覆盖率提升约20%）

**来源**：B站视频、新浪新闻、搜狐新闻
**可信度**：一手（B站视频可查证大部分）

### 3.6 家庭背景如何影响专业选择

**核心公式**：
```
越是在底层，试错成本越高 → 越要选择"确定性"
能就业、起薪高的确定性强于兴趣和理想
```

**三层分类体系**：

| 家庭类型 | 核心目标 | 推荐路径 | 避坑 |
|----------|----------|----------|------|
| **普通工薪/农村家庭** | 低风险、快回报、稳就业，尽快经济独立 | 公费师范 -> 军校/警校 -> 电气 -> 计算机/电子 -> 护理/会计 | 远离所有"虚"专业和管理类；生化环材耗不起 |
| **中产家庭** | 稳妥为主，兼顾发展空间 | 计算机/电子+复合（T型人才如"计算机+金融"）、临床5+3、自动化/电子信息（需适度深造） | 避免过于冷门或过于烧钱的方向 |
| **富裕家庭** | 兴趣优先，子承父业，拓宽视野 | 金融/经济/经管（优先名校，利用家庭资源进投行/私募）；人文社科（学术或文化路线）；艺术（有资源支撑可大胆追梦） | 限制较少 |

**张雪峰经典原话（出现在多场直播中）**：
> "普通家庭不求人，医学、军校、师范生。"
> "富裕家庭继承人，金融、经济和经管。"
> "毕业就想高薪，名校电子通信、计算机。"

**来源**：雪球网、网易、多篇媒体汇总
**可信度**：一手+二手，各来源引用高度一致

### 3.7 考研率/就业率对选择的影响

**核心判断**：张雪峰的整个方法论本质上是"考研反推高考"的产物。

- 他本人是考研辅导出身（起家于考研名师），用考研择校的精细度倒推高考志愿填报
- 传统高考填报只盯着分数和校名，他引入了专业二级学科、学科评估排名、导师研究方向、考研专业课、就业保研率等信息维度
- 他将"将来想不想考研？"作为"经典八连问"之一

**考研对填报决策的实际影响**：
- 想考研：可接受基础学科（数学、物理等）进行长线投资，本科打好基础，研究生再定方向
- 不想考研：必须选择应用型、上手快的专业（计算机、护理、电气等），否则毕业即失业

**就业率指标的使用**：
- 他频繁引用各专业/院校的公开就业数据做论证
- 对"保底志愿"的要求是"必须是就业率90%+的专业"
- 对"天坑专业"的判定标准之一就是"本科无法直接就业"

来源：B站系列视频、媒体分析
可信度：二手（其方法论确实有考研辅导出身的基因）

## 四、自创术语和核心概念

### 4.1 "专业壁垒"（核心概念，贯穿整个体系）

**定义**：一个专业对从业者的技术门槛和不可替代性。

**高壁垒专业特征**：
- 必须有硬技术，别人无法短期替代
- 不看家庭背景，凭本事吃饭
- 有职业资格认证保护（医师、律师、教师编）
- 越老越吃香，经验积累有价值
- AI难以完全取代

**低壁垒专业特征**：
- "万金油"——啥都能干啥都不精
- 是人是鬼都能干，没有准入门槛
- 容易被AI或廉价劳动力替代

### 4.2 "经典八连问"（方法论框架）

**完整版本一（面向所有考生）**：
1. **学医吗？** → 筛选医学类（临床、口腔、护理、药学等）
2. **学农吗？** → 筛选农学类（植物生产、动物医学、林学等）
3. **当老师吗？（文科可报）** → 筛选师范类
4. **学文理兼招专业吗？** → 筛选法学、新闻传播、经济管理
5. **学理工科，将来要带化学类课吗？** → 筛选化工、材料、环境、食品
6. **学理工科，将来要带物理类课吗？** → 筛选理工大类（7-8细分）
7. **带物理的话，主要是力学方向还是电学方向？** → 力学方向=土木/机械；电学方向=计算机/电子/电气/自动化
8. **想学垂直行业类专业吗？** → 筛选地质、矿业、船舶、航空航天、兵器、核工程

**"灵魂八连问"版本（含家庭/个人条件评估）**：
1. 家里有没有钱？
2. 家里有没有人？
3. 孩子成绩什么水平？
4. 男孩还是女孩？
5. 性格内向还是外向？
6. 将来想不想考研？
7. 未来想去哪个城市？
8. 对孩子最大期望是什么？

来源：B站视频、《手把手教你填报高考志愿》（2026版）目录即为八连问结构
可信度：一手

### 4.3 "天坑专业"

原指"生化环材"（生物、化学、环境、材料）。张雪峰将其推广为大众用语。他将此四类专业合称为"四大天坑"。

**核心判断标准**：本科无法直接就业，必须读到博士才有出路。

**动态修正**：张雪峰后来做过调整——如果读到名校博士且方向偏向半导体、新能源，反而成为风口。说明他对"天坑"的判断是随产业变化的，并非绝对。

### 4.4 "黄金/白银/风险/天坑"四级专业分类

| 等级 | 定义 | 示例 |
|------|------|------|
| 黄金 | 闭眼冲（技术壁垒高+就业面宽+起薪高） | 计算机、电子、电气、口腔医学、公费师范 |
| 白银 | 稳妥面广（就业稳但不暴富） | 自动化、机械（偏智能）、护理、汉语言文学 |
| 风险 | 慎报（依赖家庭背景或需名校） | 金融、经济、工商管理、旅游管理 |
| 天坑 | 尽量避开（本科就业困难） | 生化环材、历史哲学、新闻 |

### 4.5 "冲稳保"策略

非他原创，但他给出了自己的具体公式：
- **常规比例**：冲30% : 稳50% : 保20%
- **二本及以下**：冲10% : 稳50% : 保40%
- 他强调：保底志愿必须是"一定能上、专业能接受"的学校
- **冲的学校**：只冲专业组内所有专业都能接受的学校
- **稳的学校**：核心分数段，确保能录到好专业

### 4.6 "城市 > 学校 > 专业"三角模型

核心决策框架。非他首创，但他是推广这一排序最具影响力的人物。

### 4.7 "行业性院校" vs "综合性大学"

他将高校分为两类：
- **行业性院校**：电力院校（华电、东北电力、上海电力）、邮电院校（重邮、南邮）、财经院校、医学院——在对应行业内有"小圈子"优势，就业甚至强于同档综合类院校
- **综合类大学**：牌子大但专业杂，冷门专业的就业资源未必比得上行业性院校的对口专业

### 4.8 "窝里横"院校

他的自创说法：在本省认可度极高但在省外知名度低的省属重点大学。是否值得报考取决于——如果计划在本省就业，性价比很高；如果计划出省，价值打折扣。

### 4.9 "第一学历是一辈子的烙印"

核心观点：本科（第一学历）影响到一生，所以高考志愿的选择比考研选择更重要。这是张雪峰体系中最具"焦虑驱动"特性的概念。

## 七、方法论体系总览

### 7.1 决策框架流程

```
第1层：条件评估（灵魂八连问梳理个人条件）
    ↓
第2层：大类筛选（学医/学农/当老师/理工/文理兼招）
    ↓
第3层：分数匹配（分段策略：冲985 vs 保专业 vs 学手艺）
    ↓
第4层：地域锁定（结合产业布局和专业方向选城市）
    ↓
第5层：院校筛选（行业性院校 vs 综合类大学 / "窝里横"识别）
    ↓
第6层：专业排雷（黄金/白银/风险/天坑四级分类）
    ↓
第7层：填报执行（冲稳保比例、服从调剂决策、位次换算）
```

### 7.2 底层驱动逻辑

```
普通家庭 → 试错成本高 → 追求确定性
   → 确定性 = 技术壁垒（专业硬实力）
   → 技术壁垒 = 越高越好（不被替代、不看背景、越老越值钱）
   → 好城市放大技术壁垒的收益（实习机会、校招资源、人脉圈层）
   → 最终目标：能就业、起薪高、30岁不被淘汰
```

### 7.3 "三条铁律"（出处：搜狐、多源汇总）

1. **普通家庭不求人**：医学、军校、师范生
2. **富裕家庭继承人**：金融、经济和经管
3. **毕业就想高薪**：名校电子通信、计算机
