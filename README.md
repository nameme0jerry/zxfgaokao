# 🎓 张雪峰 · 高考志愿填报思维框架（Skill）

> **「张雪峰高考志愿填报方法论 · Skill。5个心智模型、10条决策启发式，帮普通家庭选专业、选学校、填志愿。」**

zxfgaokao 是一个用于自动激活张雪峰（1984-2026）高考志愿填报思维框架的 Claude Code Skill。基于 9 本著作、51 讲 B 站课程、50+ 个决策案例的深度调研，提炼出 5 个核心心智模型、10 条决策启发式和完整表达 DNA。用户只需要提出志愿填报相关的需求，Skill 会自动以张雪峰的第一人称视角，运用其就业导向、信息透明的决策框架给出建议。

它适合正在为高考志愿发愁的家长、对选专业选学校感到迷茫的高中生，以及想了解张雪峰思维方式的教育从业者。
备注：这款skill是AI工具，其回答未必准确无误，仅提供观点作为参考。

## Features

- **5 个核心心智模型**：专业壁垒论、家庭资源决定论、信息差即命运、风险规避优先、工具理性至上，每个模型标注来源证据和失效条件
- **10 条决策启发式**：覆盖选专业、选学校、服从调剂、冲稳保策略、考公方向、分数段应对等高频场景
- **完整的表达 DNA 还原**：短句暴击、反问推进、类比扎心、东北口音直爽感，读 100 字能认出是张雪峰
- **内置 Agentic Protocol**：遇到需要事实的问题先搜索再回答，不凭训练数据编造
- **诚实边界声明**：标注言论时效性、公开与真实差距、反身性困境等局限，不自欺欺人

## Decision Heuristics

- **普通家庭不碰金融**——家里能存 1000 万以上再考虑
- **理工重专业，文重学校**——理科专业 > 学校 > 城市；文科学校 > 专业 > 城市
- **服从调剂铁律**——普通批次必须勾选，不勾选 = 退档
- **考公四大王牌**——汉语言、法学、会计、思政
- **生化环材门槛**——不读硕博不要碰
- **医科地域原则**——想在哪儿当医生，就读哪儿的医学院
